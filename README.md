## Serverless to AWS lambda

### download serverless

```shell
$  npm i -g serverless
```

### Set AWS access key into

```shell
$  aws login ...
```

### Build project

```shell
$  yarn build
```

### Pckaging into serverless

```shell
$  npx serverless package
```

### Deploy

```shell
$  npx serverless deploy
```
